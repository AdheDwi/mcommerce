# DheBoilerplate

Dhe Next.js 8 Boilerplate

Prerequisites

Node.js (v10 or higher)

npm (v6.5 or higher)

What's Inside?

Next.js (React.js with superpower)

Custom server (Express)
Custom routing (next-routes)
Custom http request wrapper (apisauce)

Redux (saga)
CSS

Getting Started

Development

# Clone the repo and then run

$ npm install
  $ npm run dev

Production

# Clone the repo and then run

$ npm install
  $ npm run build
\$ npm start
